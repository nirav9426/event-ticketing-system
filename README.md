# Event Ticketing System Backend

A simple backend for an event ticketing system built with Python.

## Features
- User authentication
- Event management
- Ticket booking
- Payment processing

## Technologies Used
- Python
- Flask
- SQLAlchemy
- PostgreSQL

## Setup
1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Run migrations
5. Start the server

## Usage
- Register users
- Create events
- Book tickets
- Process payments
